Ordering slides

  • You can grab any slide and drag it to left or right on the slides tab.

      • Slide options

      • Image This will be the image of the slide, it is a mandatory field.
      • Thumbnail This will be the thumbnail of the slide. If you leave this field empty, Cute Slider will be use the slide image if the thumbnail navigation is enabled.
      • Delay The delay in seconds between the current and the next slide, the slideshow feature will wait the time specified here.

      Transitions

      Here you can select your transitions list which the slider will use on this slide. If you select more than one transition, the slider will choose a random one from your selection. If you don't select any transition, a random one will be used of all the available transitions.

      Link this slide

      • URL If you want to link the whole slide, you can specify an URL here
      • Target The target property of the link, _blank means new tab/page.

      Info box

      You can create easily an information box with your custom title, text, link, etc.

      • Alignment The aligment of the info box, for example bottom or left.
      • Title The title of the info box if any.
      • Text The contents of the info box if any. Can contain HTML code.
      • Link name If you specify an URL, a button will appear in the info box. Here you can add your custom button text.
      • Link URL The URL of the infobox link/button
      • Link target The target property of the link, _blank means new tab/page.

      Misc

      • Video URL You can create a fullscreen video in the slide with filling this field. Please note that you have to enter the embed URL of the video, this can be found in the embed code in the video page on Youtube or Vimeo.
      • Hidden If you are working on a slide, but you don't want to publish it until it is finished, you can temporarly hide with this checkbox.
