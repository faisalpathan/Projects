Naming captions

  • To add a name to a caption just simply click on its default name and type the new one.

      • HTML tab

      This will be the content of your caption. Of cource, you can use any HTML code in this field.

      Options tab

      • Effect The effect of the caption. Can be fade or slide.
      • Delay If you specify a delay time here in millisecs, your caption won't appear immidiately, you can postpone it relative to the other captions.

      Link tab

      • URL You can link your captions, just enter your URL into this text field.
      • Target The target property of the link, _blank means new tab/page.

      Style tab

      Here you can set the appearance of your caption, such as the background color, its size and font type.
