Language support

Cute Slider WP is translation and localization ready. If you want to tranlate the entire plugin to your own language including the admin area, you can find all the details in the documentation that came with the package that you downloaded from CodeCanyon. For translating sliders, see the below section.

Translating sliders

Cute Slider WP works together with translation plugins like qTranslate seamlessly. In the corresponding text fileds (for example the caption HTML field) you can use the shortcode of your translation plugin.

qTranslate example

qTranslate a widely known plugin for translate your WP content. Below there is an example to how to use qTranslate if you choose this plugin for your site.

Example: [:pt]portuguese text[:en]english text
